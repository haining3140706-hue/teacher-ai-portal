# AI教学知识库平台

> 面向酒店管理专业的 AI 智能教学平台 —— 桌面端单页应用

基于 React 18 + Tailwind CSS 构建的酒店管理专业 AI 教学知识库平台，集成知识库管理、AI 课件生成、任务分发引擎、全息学情分析等核心功能模块，为专业教师提供从备课到分发到学情分析的全链路智能辅助。

## 功能架构

### 🖥️ 工作台总览
- 四维数据看板：授课班级、学生人数、知识库资源、高频预警点
- AI 教学助手建议：自动识别薄弱模块并推荐干预方案
- 快捷入口面板：一键跳转核心功能
- 课程教学进度与最近动态一览

### 📚 专业知识库
- 多类型资源管理：教材语料(doc)、真实案例(case)、视频多媒体(video)、智能题库(quiz)
- 五大筛选维度：全部资源 / 专业教材语料 / 行业真实案例 / 视频 / 智能题库
- 基于 RAG 的全库检索
- 资源详情弹窗：评分、下载量、标签、一键用于备课或分发

### ✨ AI 备课中心
- 自然语言提示词驱动课件生成
- 预设示例模板：客房清扫 SOP / 超售客诉处理 / 餐饮摆台实操
- 结构化活页输出：情景导入 → 理论讲解 → 角色扮演 → 随堂测验
- 支持导出 PPT 与保存到我的课件

### 📤 任务分发引擎
- 三步引导式分发流程：选择任务 → 配置对象 → 规则设置
- AI 差异化分发建议：自动识别基础薄弱学生，附加强化内容
- 防作弊设置：选项随机化、题目随机化、切屏警告
- 时间控制：开放时间、截止时间、限时设置

### 📊 全息学情分析
- 核心知识点掌握率雷达图（SVG 绘制）
- 近七日学习活跃度与任务提交柱状图
- AI 学情智能诊断：薄弱项预警 + 重点关注名单
- 一键生成补救任务 / AI 备课强化

### 🎓 课程管理
- 课程列表与状态（进行中 / 已结课）
- 课程详情弹窗：在读学生数、关联班级、教学进度
- 快捷操作：AI 备课 / 查看学情 / 分发任务

### 👥 学生管理
- 班级筛选 + 姓名搜索
- 学情状态标签：预警 / 正常
- 成绩趋势：↑上升 / ↓下降 / →稳定
- 学生详情弹窗：各模块成绩条形图 + AI 补救方案生成

### 🤖 AI 教学助手
- 右侧抽屉式对话面板
- 4 大快捷指令：生成学情报告 / 推荐教学案例 / 设计随堂测验 / 制定补救方案
- 上下文感知回复，支持跳转对应功能页面

### ⚙️ 系统设置
- 个人信息管理
- 通知设置（学情预警 / 任务完成 / AI 生成 / 系统更新）
- AI 参数配置：模型选择（DeepSeek-V3 / GPT-4o / Qwen-Max）、Temperature、评分严格度、RAG 检索范围
- 安全与隐私：密码修改、两步验证、学生数据匿名化

## AI 能力配置

平台支持三种大模型接入：

- **DeepSeek-V3**（推荐）—— 中文场景优化，性价比高
- **GPT-4o** —— 多模态能力强
- **Qwen-Max** —— 通义千问旗舰模型

可配置参数：
- Temperature（创造力参数）：0~100
- AI 评分严格度：宽松 / 标准 / 严格
- RAG 知识库检索范围：全库 / 仅当前课程 / 仅教材语料

## 自定义与扩展

### 修改知识库资源

编辑 `resourceLibrary` 数组：

```js
const resourceLibrary = [
  {
    id: 7,
    type: 'case',           // case | video | quiz | doc
    title: '你的资源标题',
    tags: ['标签1', '标签2'],
    desc: '资源描述...',
    downloads: 0,
    rating: 4.5
  },
];
```

### 添加课程

向 `courseList` 数组追加：

```js
const courseList = [
  { id: 5, name: '新课程名称', code: 'HTM-401',
    classes: ['24级酒管1班'], students: 0,
    progress: 0, status: 'active' },
];
```

### 添加 AI 对话预设

编辑 `aiChatPresets` 数组：

```js
const aiChatPresets = [
  { label: '📋 你的指令', prompt: '完整的提示词文本' },
];
```
